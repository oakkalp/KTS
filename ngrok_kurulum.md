# ngrok ile HTTPS (Daha Kolay)

## 1. ngrok İndir
https://ngrok.com/download

## 2. ngrok'u Çalıştır
```
ngrok http 80
```

## 3. Verilen HTTPS URL'yi Kullan
```
https://abc123.ngrok.io/kuryefullsistem/kurye/dashboard.php
```

## Avantajları:
- ✅ Anında HTTPS
- ✅ Kurulum gerektirmez  
- ✅ Dışarıdan erişilebilir
- ✅ Gerçek SSL sertifikası

## Dezavantajları:
- ⚠️ Her açılışta farklı URL
- ⚠️ İnternet bağlantısı gerekli
